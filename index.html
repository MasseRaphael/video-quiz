Information importante!

Un système de questionnaire est en place sur la vidéo.

De ce faite si vous devez mettre en pause la vidéo utiliser le bouton sur la droite afin de suspendre le timer, et avoir vu tout les éléments nécéssaire pour répondre aux questions.

Assurez-vous que le timer sous la vidéo soit le plus synchronisé possible avec le temps de la vidéo. (1 à 2 secondes sont acceptable)
Si ce n'est pas le cas suspendez la vidéo avec le bouton pause puis lancez la manuellement jusqu'à rattraper le compteur puis cliquer sur le bouton play.
